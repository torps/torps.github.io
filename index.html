The Tor Path Simulator (TorPS)

TorPS quickly simulates path selection in Tor. The code can be obtained with

git clone https://github.com/torps/torps.git

TorPS was used to produce results for the paper
Users Get Routed: Traffic Correlation on Tor by Realistic Adversaries
by Aaron Johnson, Chris Wacek, Rob Jansen, Micah Sherr, and Paul Syverson
To appear in Proceedings of the 20th ACM Conference on Computer and Communications Security (CCS 2013).
